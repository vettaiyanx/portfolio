# Suriya P - 3D Animated Portfolio

A premium, immersive 3D portfolio built with Three.js, CSS Glassmorphism, and Node.js.

## 🚀 Live Demo
- **Frontend**: [suriyaportfolio-five.vercel.app](https://suriyaportfolio-five.vercel.app)
- **Backend API**: [portfolio-00jv.onrender.com](https://portfolio-00jv.onrender.com)

## ✨ Features
- **3D Interactive Background**: High-performance Three.js particles and rotating geometries.
- **Glassmorphism UI**: Modern, frosted-glass design system.
- **Dynamic Typing**: Animated hero text for professional roles.
- **Interactive Stats**: Counting animations for career milestones.
- **Contact Integration**: Live Node.js/Express backend for form submissions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript, Three.js
- **Backend**: Node.js, Express.js
- **Deployment**: Vercel (Frontend), Render (Backend)

## 📂 Structure
- `/frontend`: Static assets, Three.js logic, and UI.
- `/backend`: Express server and contact API.
